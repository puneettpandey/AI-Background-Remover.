# 🚀 AI Background Remover

A simple web app that removes image backgrounds using AI.

---

## ✨ Features
- Upload image  
- Remove background using AI  
- Preview result  
- Download transparent image  

---

## 🛠 Tech Stack
HTML | CSS | JavaScript | Fetch API  

---

## ⚡ Why This Project?
Demonstrates API integration, file handling, and asynchronous JavaScript in a real-world use case.

---

## 🚀 Run Locally
1. Download or clone the repo  
2. Open `index.html` in browser
